# kii-readiness-mapper

> [!IMPORTANT]
> Этот инструмент предназначен **только для самооценки** и не заменяет официальное категорирование или юридическую консультацию.

## Что это

**kii-readiness-mapper** — это инструмент самооценки для организаций, которые хотят понять:
* подпадают ли они под понятие субъекта КИИ (Федеральный закон №187-ФЗ);
* какие их системы потенциально могут стать объектами КИИ;
* какие организационные и технические меры уже реализованы, а где есть «дыры».

## Почему это важно

187-ФЗ обязывает субъекты КИИ:
* идентифицировать объекты;
* провести категорирование;
* выполнить комплекс мер по защите.

Многие компании (особенно SMB и региональные) часто не понимают, относятся ли они к КИИ, и с чего начать. Этот инструмент помогает структурировать информацию.

## Как использовать (пример)

```bash
pip install kii-readiness-mapper

# 1. Сгенерировать профиль
kii-readiness init-profile --output org-profile.yaml

# 2. Заполнить или собрать ответы через CLI-анкету
kii-readiness run-cli-survey --sector health --system-type web --output answers.yaml

# 3. Выполнить оценку
kii-readiness assess-profile --profile org-profile.yaml --answers answers.yaml --out result.json

# 4. Сгенерировать отчёт
kii-readiness generate-report --result result.json --format markdown --output report.md
```

## Профессиональные услуги – run-as-daemon.ru

Проект развивается инженером DevOps/DevSecOps с сайта [run-as-daemon.ru](https://run-as-daemon.ru).

Если вам нужно:
- провести предаудит на предмет КИИ и 187-ФЗ;
- подготовиться к формализованному обследованию и категорированию;
- спроектировать набор орг- и тех. мер и подобрать сертифицированные средства защиты,

вы можете заказать консалтинг, обследование и сопровождение.

## Важные оговорки

Инструмент:
* **не заменяет** официальное категорирование и обследование;
* **не является** юридическим заключением;
* **не гарантирует** соответствия требованиям регуляторов.

Результаты нужно воспринимать как ориентир для подготовки к работе с лицензированными организациями и консультантами.
