# kii-readiness-mapper

> [!IMPORTANT]
> This tool is for **self-assessment only** and does not replace official categorization or legal advice.

## English

**kii-readiness-mapper** is a tool for self-assessment of readiness for Russian Critical Information Infrastructure (CII) status under Federal Law 187-FZ.

It provides:
* A questionnaire (CLI/Web) tailored to various sectors.
* A JSON/YAML profile builder for your organization and systems.
* A report generator estimating the probability of being a CII subject and identifying security gaps.

**Note:** This is not an official audit tool. It is a helper for the preparatory stage.
